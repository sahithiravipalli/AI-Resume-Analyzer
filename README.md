# AI Resume Analyzer
A professional resume analysis dashboard built using Python and Streamlit.
## Features
- ATS Score Calculation
- Skill Detection
- Resume Summary
- Improvement Suggestions
- Job Role Matching
- Download Report
- SQLite Database Storage
